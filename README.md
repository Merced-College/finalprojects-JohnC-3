Project Title: Library Manager

Description: This program allows the user to use provided tools to manage library resources. To run the program, simply click the run button and follow the prompts given. This usually consists of inputting a number correlated with the desired feature or typing needed information to complete the process. The features of this program consists of the following: search for a book, get a list of all books, search for a member, get a list of all members, check out a book to a member, allow a member to check in a book, let a member reserve/queue for a book, add and remove a book, and add and remove a member. All of these tools allow users to quickly and efficiently access information from the library, as well as add or remove information when needed.

Data Structures Used: Arrays, ArrayLists, and Queue

Algorithms Used: Binary Search, Insertion Sort, and Loading algorithms
